# 03 — Painel do Gestor: Visão Geral

## O que é o Painel do Gestor

O **Painel do Gestor** é a principal ferramenta de trabalho do perfil gestor. Ele permite visualizar todas as informações da plataforma em modo de somente leitura.

### Título do Painel

> **Painel do Gestor**
> _Visualize materiais, usuários, trilhas e métricas da plataforma._

## Abas Disponíveis

O painel possui **4 abas** para navegar:

| Aba | Ícone | Descrição |
|---|---|---|
| **Materiais** | 🖼️ | Lista de todos os materiais da plataforma |
| **Usuários** | 👥 | Lista de todos os usuários cadastrados |
| **Trilhas** | 📖 | Coleções de materiais organizados |
| **Métricas** | 📊 | Indicadores de desempenho e rankings |

### Indicador Visual de Aba Ativa

A aba ativa fica destacada com efeito **liquid-glass-gold** (fundo dourado translúcido).

## O que o Gestor NÃO Pode Fazer

⚠️ **Importante:** O gestor tem acesso **somente leitura**. Você **não verá** botões de:

- "Criar"
- "Editar"
- "Excluir"
- "Exportar"

Essas ações são exclusivas do Administrador.

## Acesso ao Painel

1. No menu lateral, clique em **Painel de Gestão**
2. Você será direcionado para a aba de Materiais por padrão
3. Use as abas para navegar entre as diferentes visualizações

## Dicas de Uso

1. **Use filtros** para encontrar informações rapidamente
2. **Navegue pelas abas** para ter visão completa
3. **Acesse métricas** para acompanhar desempenho
4. **Clique em itens** para ver detalhes (ex: trilhas, materiais específicos)
