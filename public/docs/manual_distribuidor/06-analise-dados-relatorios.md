# 06 — Análise de Dados & Relatórios

## Decisões Baseadas em Evidências

Para um Distribuidor de sucesso, a intuição deve ser validada por dados. O Conexão Hub oferece um conjunto de relatórios exportáveis que transformam a atividade da plataforma em insights de negócio.

## Relatórios Exportáveis (CSV/Excel)

Você pode extrair dados em tempo real para análises externas:

1. **Relatório de Conclusão por Turma:** Útil para verificar quem está apto para novas certificações.
2. **Ranking de XP Regional:** Base para campanhas de premiação e incentivo.
3. **Logs de Acesso:** Identifique padrões de comportamento (quais dias da semana a equipe estuda mais?).

## Como Usar os Dados para Vender Mais

- **Cruzamento de Dados:** Se a venda de um produto específico caiu na região, verifique o relatório de estudos. Talvez a equipe não tenha assistido ao treinamento técnico desse produto?
- **Identificação de "Gaps" de Conhecimento:** Se muitos consultores falham nos quizzes de um tema, organize um reforço imediato.

## Reuniões de Performance

Utilize os prints do dashboard e os dados dos relatórios em suas reuniões mensais:
- Mostre a evolução da região.
- Destaque os Top 3 consultores.
- Estabeleça metas de estudo para o próximo mês.
