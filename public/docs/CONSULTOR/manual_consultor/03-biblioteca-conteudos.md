# 03 — Biblioteca de Conteúdos

## Visão Geral

A **Biblioteca** é onde você acessa todos os materiais e recursos disponíveis na plataforma. Os conteúdos são organizados por categorias e podem ser filtrados para facilitar a busca.

## Como Acessar

1. No menu lateral, clique em **Biblioteca**
2. Você verá a lista de todos os materiais disponíveis
3. Use os filtros para refinar sua busca

## Categorias de Conteúdo

Os materiais são organizados nas seguintes categorias:

| Categoria | Descrição |
|-----------|------------|
| **Manuais** | Documentação técnica e guias de uso |
| **Vídeos** | Tutoriais em vídeo |
| **Apresentações** | Slides e materiais visuais |
| **Downloads** | Arquivos para baixar |
| **Artigos** | Conteúdos textuais educacionais |

## Sistema de Busca

### Busca por Texto

Digite no campo de busca para encontrar materiais por:

- Título do material
- Descrição
- Palavras-chave

### Filtros Disponíveis

| Filtro | Função |
|--------|--------|
| **Categoria** | Selecionar tipo de material |
| **Data** | Ordenar por mais recentes ou antigos |
| **Tipo de arquivo** | PDF, Vídeo, Imagem, etc. |
| **Relevância** | Ordenar por popularidade |

## Visualização de Materiais

### Ao clicar em um material

1. Abre-se uma página de visualização
2. Você pode ler o conteúdo diretamente
3. Ou clicar em "Baixar" para salvar no seu dispositivo

### Formatos Suportados

- **PDF**: Visualização inline e download
- **Vídeo**: Player nativo com controles
- **Imagens**: Galeria com zoom
- **Apresentações**: Visualizador de slides

## Favoritos

### Salvar Material como Favorito

1. Abra o material desejado
2. Clique no ícone de estrela (favorito)
3. O material será salvo na sua lista de favoritos

### Acessar Favoritos

1. No seu perfil, vá para **Meus Favoritos**
2. Veja todos os materiais salvos
3. Clique para acessar diretamente

## Histórico de Leitura

### O que é registrado

A plataforma registra automaticamente:

- Materiais visualizados
- Tempo de leitura
- Data do último acesso

### Acessar Histórico

1. Vá para **Meu Perfil**
2. Clique em **Histórico**
3. See todos os materiais já acessados

## Dicas para Usar a Biblioteca

1. **Use filtros** para encontrar exatamente o que precisa
2. **Salve favoritos** para acesso rápido posterior
3. **Verifique o histórico** para revisar materiais importantes
4. **Explore categorias** para descobrir novos conteúdos
