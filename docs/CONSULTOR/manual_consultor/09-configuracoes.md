# 09 — Configurações

## Menu de Configurações

No menu lateral, clique em **Configurações** para acessar suas preferências pessoais.

## Preferências de Idioma

### Idiomas Disponíveis

A plataforma suporta 3 idiomas:

| Código | Idioma | Bandeira |
|--------|--------|----------|
| `pt-br` | Português (Brasil) | 🇧🇷 |
| `en-us` | English (US) | 🇺🇸 |
| `es-es` | Español | 🇪🇸 |

### Como Alterar o Idioma

1. Nas configurações, vá para **Idioma**
2. Selecione o idioma desejado
3. A interface será atualizada imediatamente

### Alteração Rápida pelo Header

Você também pode alterar o idioma diretamente pelo header:

1. No topo da página, clique no seletor de idioma
2. Escolha a opção desejada (PT, EN, ES)
3. A mudança é aplicada instantaneamente

## Configurações de Notificações

### Tipos de Notificação

| Notificação | Descrição |
|-------------|------------|
| **Promoções de nível** | Quando você sobe de patente |
| **Novos badges** | Quando você conquista um badge |
| **Conclusão de trilha** | Quando você completa uma trilha |
| **Lembretes de progresso** | Sugestões de continuidade |
| **Novos conteúdos** | Quando há novos materiais na biblioteca |

### Gerenciar Notificações

1. Nas configurações, vá para **Notificações**
2. Ative ou desative cada tipo de notificação
3. Su preferências são salvas automaticamente

### Notificações por Email

| Opção | Descrição |
|-------|------------|
| **Instantâneo** | Receba imediatamente |
| **Resumo diário** | Receba um resumo por dia |
| **Resumo semanal** | Receba um resumo por semana |
| **Desativado** | Não receba por email |

## Configurações de Exibição

### Tema Visual

A plataforma opera em **modo escuro permanente** (dark mode). Não há opção de tema claro.

### Densidade de Informação

| Opção | Descrição |
|-------|------------|
| **Completo** | Mostra todos os detalhes |
| **Compacto** | Visual mais organizado |

## Preferências de Aprendizado

### Trilhas Recomendadas

Você pode configurar quais tipos de trilhas deseja ver recomendados:

- Por nível de dificuldade
- Por área de interesse
- Por duração estimada

### Progresso Visível

Escolha se deseja ver:

- Barra de progresso nas trilhas
- Porcentagem de conclusão
- Tempo estimado restante

## Dados e Privacidade

### Exportar Dados

Você pode exportar seus dados pessoais:

1. Vá para **Privacidade**
2. Clique em **Exportar meus dados**
3. Receba um arquivo com todas as suas informações

### Histórico de Atividades

Você pode visualizar e excluir seu histórico:

1. Vá para **Privacidade**
2. Clique em **Histórico de atividades**
3. Selecione atividades para remover

## Segurança

### Sessões Ativas

Visualize onde você está logado:

1. Vá para **Segurança**
2. Veja lista de dispositivos/sessões ativas
3. Encerre sessões indesejadas

### Alteração de Senha

Para alterar sua senha:

1. Vá para **Segurança**
2. Clique em **Alterar senha**
3. Digite senha atual e nova senha
4. Confirme a alteração

## Salvar Alterações

Todas as alterações em configurações são salvas automaticamente. Você verá uma confirmação visual ao fazer mudanças.
