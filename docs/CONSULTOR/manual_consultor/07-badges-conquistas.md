# 07 — Badges e Conquistas

## O que são Badges

Os **Badges** são conquistas visuais que você obtém ao atingir marcos específicos na plataforma. Cada badge representa uma realização e fica exposto na sua galeria de conquistas.

## Como Desbloquear um Badge

### Processo Automático

1. Realize a atividade requerida
2. O sistema verifica automaticamente
3. Se critérios atendidos, badge é desbloqueado
4. Notificação informa sobre nova conquista

### Visualizar Badge

1. Vá para **Conquistas** no menu
2. See todos os badges (desbloqueados e bloqueados)
3. Badges bloqueados aparecem "travados" com ícone de questão

## Galeria de Conquistas

### Acesso

1. Clique em **Conquistas** no menu lateral
2. See a galeria completa de badges

### Organização

Os badges são organizados em abas:

- **Todos**: Todos os badges
- **Desbloqueados**: Apenas os conquistados
- **Em Progresso**: Próximos a desbloquear
- **Secretos**: Badges com critérios ocultos

### Detalhes do Badge

Ao clicar em um badge, você vê:

- Nome do badge
- Descrição
- Critério para desbloquear
- Data de conquista (se desbloqueado)

## Critérios Secretos

Alguns badges possuem critérios ocultos — você sabe que existem mas não sabe como desbloquear. Descubra completando atividades e explorando a plataforma!

## Dicas para Conquistar Badges

1. **Mantenha consistência** — Badges de sequência dependem de acesso regular
2. **Explore tudo** — Alguns badges estão em funcionalidades menos óbvias
3. **Complete trilhas** — Maior fonte de badges
4. **Leia materiais** — Accumulate de badges de leitura
