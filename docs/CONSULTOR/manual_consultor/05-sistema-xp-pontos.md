# 05 — Sistema de XP e Pontos

## O que é XP

**XP (Experience Points)** é a moeda de progressão da plataforma. Cada atividade realizada na plataforma concede XP, que accumulation para você subir de nível e conquistar novas patentes.

## Como Ganhar XP

### Atividades que Geram XP

| Atividade | XP Concedido |
|-----------|--------------|
| Ler um material completo | 10 XP |
| Completar um módulo de trilha | 25 XP |
| Finalizar uma trilha completa | 100 XP |
| Responder checkpoint correto | 15 XP |
| Atingir 100% em uma trilha | 50 XP (bônus) |
| Acessar a plataforma diariamente | 5 XP |
| Completar primeira trilha do mês | 30 XP (bônus) |

### Bônus de Desempenho

| Conquistas | Bônus XP |
|------------|----------|
| Primeira trilha do dia | +20% XP |
| 3 trilhas na mesma semana | +50 XP |
| Mês com acesso diário | +100 XP |

### Pontos por Interação

| Tipo de Interação | XP |
|-------------------|----|
| Favoritar material | 2 XP |
| Comentar em material | 5 XP |
| Compartilhar trilha | 10 XP |

## Histórico de Pontos

### Acessar Histórico

1. No seu perfil, vá para **Estatísticas**
2. Clique na aba **Histórico de XP**
3. See todos os ganhos de XP por data

### Detalhes do Histórico

O histórico mostra:

- Data da atividade
- Tipo de atividade
- XP ganho
- XP total acumulado

## Como Acompanhar seu XP

### No Header

Seu XP total é exibido no header no formato:

```
{NÍVEL} · {PONTOS} XP
```

Exemplo: `BRONZE · 150 XP`

### No Dashboard

O card de "Seu Progresso" mostra:

- Ponto atual no nível
- Progresso até próximo nível (barra)
- Quantidade de XP necessária para próxima promoção

## Dicas para Ganhar XP Eficientemente

1. **Mantenha constância** — Acesso diário gera XP mesmo sem atividades grandes
2. **Complete trilhas** — Maior fonte de XP por atividade
3. **Use bônus** — Fique atendo aos bônus por desempenho
4. **Leia materiais completos** — XP extra por conclusão total
