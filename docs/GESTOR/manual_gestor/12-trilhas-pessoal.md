# 12 — Trilhas Pessoais

## O que são Trilhas

As **Trilhas** são percursos estruturados de aprendizado que guiam você através de materiais específicos, com objetivo de certificar seu conhecimento em determinado tema.

Cada trilha é composta por:

- **Módulos**: Unidades de conteúdo
- **Materiais**: Vídeos, documentos, quizzes
- **Avaliações**: Testes de conhecimento
- **Certificação**: Diploma ao completar

## Visualizar Trilhas Disponíveis

1. No menu lateral, clique em **Trilhas**
2. See a lista de todas as trilhas disponíveis
3. Cada trilha mostra:
   - Título e descrição
   - Número de módulos
   - Tempo estimado de conclusão
   - Nível de dificuldade

## Como Iniciar uma Trilha

1. Na lista de trilhas, clique na desejada
2. Leia a descrição e objetivos
3. Clique em **Iniciar Trilha**
4. Comece pelo primeiro módulo

## Progresso e Acompanhamento

### Barra de Progresso

Cada trilha mostra uma barra de progresso indicando:

- Módulos concluídos
- Tempo investido
- Próximo paso

### Checkpoints

Ao final de cada módulo, você encontra um checkpoint para testar o conhecimento.

## Certificações

### Obtenção do Certificado

Para receber o certificado de uma trilha, você deve:

1. Concluir todos os módulos
2. Atingir 70% ou mais nos checkpoints
3. Responder o questionário final

### Visualizar Certificados

1. Vá para **Meu Perfil**
2. Clique em **Certificados**
3. See todos os certificados obtidos
