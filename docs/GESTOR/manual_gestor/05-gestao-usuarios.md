# 05 — Gestão de Usuários

## Visualizar Usuários

Na aba **Usuários** do Painel do Gestor, você pode visualizar todos os usuários cadastrados na plataforma.

### Barra de Filtros

1. **Campo de busca** — Busca por nome ou email do usuário.
2. **Filtro por perfil** — Dropdown com:
   - Todos
   - Cliente
   - Distribuidor
   - Consultor
   - Gestor

## Tabela de Usuários

### Desktop

| Coluna | Descrição |
|---|---|
| **Usuário** | Nome + email + ícone de avatar |
| **WhatsApp** | Número de telefone (com link clicável) |
| **CRO** | Registro profissional (quando preenchido) |
| **Perfil** | Badge com o nome do perfil |
| **Status** | Badge colorido: Ativo, Pendente, Inativo, Recusado |
| **XP** | Pontos acumulados pelo usuário |

### Celular

No celular, cada usuário é exibido como um **card** compacto.

### Ordenação

A lista é **ordenada alfabeticamente** pelo nome.

## Status dos Usuários

| Status | Cor | Significado |
|---|---|---|
| **Ativo** | 🟢 Verde | Conta aprovada e em uso |
| **Pendente** | 🟡 Amarelo | Aguardando aprovação do admin |
| **Inativo** | ⚪ Cinza | Conta desativada |
| **Recusado** | 🔴 Vermelho | Cadastro recusado pelo admin |

## Perfis de Usuário

| Perfil | Descrição |
|--------|------------|
| **Cliente** | Usuário final que consome conteúdos |
| **Distribuidor** | Parceiro comercial |
| **Consultor** | Consultor da plataforma |
| **Gestor** | Supervisor com acesso ao painel |
| **Admin** | Administrador do sistema (não listado) |
